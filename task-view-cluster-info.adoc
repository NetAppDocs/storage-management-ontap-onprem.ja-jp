= クラスター情報と契約の詳細を表示する
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
NetApp Consoleで、オンプレミスの各ONTAPクラスターの契約詳細を表示できます。クラスターをまだ検出していない場合は、[サブスクリプションとライセンス] ページから検出することもできます。

.必要なコンソールロール:
ストレージ管理者またはストレージ閲覧者。link:https://docs.netapp.com/us-en/console-setup-admin/reference-iam-predefined-roles.html["NetApp Consoleのアクセス ロールについて学習します。"^]
